# AI Agents & Prompt Architecture for Local Content

This repository contains production-ready system prompts and custom AI agent workflows designed to streamline digital content production, copywriting, and video editing post-production.

## 🚀 Architectures Included
- **Content Strategist & Copywriter Agent:** Tailored for localized audience retention, strict structural constraints, and organic engagement growth.
- **CapCut Post-Production Director:** An operational gatekeeper agent built to sync script text with available b-roll assets and standard video effects.

*Note: The core prompts are written in European Portuguese (PT-PT) as they are deployed in production for localized regional social media projects and accessibility advocacy in Portugal.*
